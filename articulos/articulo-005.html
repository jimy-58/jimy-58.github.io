Curso de Excel aplicado a procesos administrativos y técnicos

Autor: Ing. Jaime Hernández Hernández
Publicado: 17/03/2026

Este curso está diseñado con base en necesidades reales detectadas en entornos laborales, enfocándose en la optimización de procesos administrativos, técnicos y de análisis de información mediante el uso de Microsoft Excel.

Objetivo del curso

Desarrollar en el participante las competencias necesarias para utilizar Excel de manera eficiente en la captura, análisis y automatización de información, mediante el uso de herramientas como funciones, tablas dinámicas y macros, aplicadas a contextos reales de trabajo.

Diagnóstico de necesidades

A partir de un análisis aplicado a personal administrativo y técnico, se identificaron las siguientes áreas de mejora:

  • Uso limitado de funciones lógicas
  • Dificultad en tablas dinámicas
  • Falta de automatización
  • Procesos manuales repetitivos
  • Necesidad de dashboards e informes visuales

Estructura del curso

Módulo 1: Fundamentos y estructura de datos

Organización de información, tipos de datos y creación de tablas estructuradas para un manejo eficiente.

Módulo 2: Fórmulas y funciones esenciales

Uso de funciones como SUMA, SI, BUSCARV, INDICE y COINCIDIR para resolver problemas reales.

Módulo 3: Análisis de datos

Aplicación de filtros avanzados, tablas dinámicas y segmentación para interpretar información.

Módulo 4: Visualización de datos (Dashboards)

Creación de gráficos y paneles interactivos para la toma de decisiones.

Módulo 5: Automatización con macros

Introducción a macros para optimizar procesos repetitivos y mejorar la productividad.

Metodología

El curso se desarrolla bajo un enfoque práctico, basado en la resolución de problemas reales, combinando teoría y práctica en una proporción 20% - 80%.

  • Ejercicios guiados
  • Casos reales
  • Aprendizaje progresivo

Evaluación

La evaluación se realizará mediante actividades prácticas por módulo y un proyecto final.

  • Ejercicios por módulo
  • Desarrollo de dashboard
  • Automatización de proceso real
