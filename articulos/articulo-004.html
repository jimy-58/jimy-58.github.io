Curso básico de Excel

Autor: Ing. Jaime Hernández Hernández
Publicado: 14-01-2026

Este curso básico de Excel está diseñado para personas que desean crear tablas más técnicas, profesionales y funcionales utilizando herramientas nativas de Excel, validaciones de datos y formatos personalizados.

El objetivo es que el participante no solo capture información, sino que construya hojas de cálculo estructuradas, claras y listas para procesos administrativos, técnicos o de análisis.

Dar formato a una tabla

  1. Selecciona toda la tabla (filas y columnas).
  2. Ve al menú Inicio.
  3. Selecciona Dar formato como tabla.
  4. Elige el estilo deseado (claro, medio u oscuro).
  5. Confirma la selección de la tabla.
  6. Marca la opción La tabla tiene encabezados.

Este formato permite trabajar con filtros, fórmulas estructuradas y facilita la lectura de la información.

Asignar nombre a una tabla

  1. Selecciona cualquier celda de la tabla.
  2. Ve al menú Diseño de tabla.
  3. Ubica el campo Nombre de la tabla.
  4. Renombra la tabla.

Ejemplos:

  • tblReporte
  • TablaVentas
  • t_Usuarios

Asignar nombres es fundamental cuando se trabajan múltiples tablas o se realizan relaciones y análisis más avanzados.

Validación de datos: Lista

  1. Selecciona las celdas a validar.
  2. Ve a Datos → Validación de datos.
  3. En Permitir, selecciona Lista.
  4. En Origen, escribe las opciones separadas por coma.

Ejemplo:

Planeada,En proceso,Completada

Puedes personalizar los mensajes de entrada y error para guiar al usuario y evitar capturas incorrectas.

Validación de datos: Número entero

  1. Selecciona las celdas.
  2. Ve a Datos → Validación de datos.
  3. Selecciona Número entero.
  4. Define la condición (ejemplo: menor que 100).

Este tipo de validación evita valores fuera de rango y mantiene consistencia en los datos.

Formato personalizado tipo Check (1 = Sí / 0 = No)

Para representar valores binarios de forma visual:

  1. Selecciona las celdas.
  2. Inicio → Formato de número → Más formatos.
  3. Categoría: Personalizada.
  4. Escribe:
[Verde][=1]✔;[Rojo][=0]❌

Este formato mejora la lectura visual y puede adaptarse a distintos contextos.

Recomendaciones finales

  • Guardar el archivo como habilitado para macros si se usarán.
  • Renombrar las hojas.
  • Aplicar formato a las tablas.
  • Asignar nombres claros a cada tabla.
